# ⚖️ AI 裁员谈判专家 (AI Severance Negotiator)
在算法面前，每一个职场人都值得拥有一面平等的盾牌。

## 🌟 1. 这个 Agent 能干嘛？

**方案深度评估**：分析补偿金合理性，戳穿公司“打折赔偿”的谎言。

**模拟实战谈判**：让 AI 扮演“冷酷 HR”或“强硬法务”，陪你在进会议室前练好话术。

**风险雷达监测**：识别公司是否存在违法解除（如三期女职工、医疗期非法辞退等）。

**证据收集指南**：手把手教你如何合法取证，保护自己。

## 💡 2. 为什么要做这个 Agent？

在面临裁员或劳动纠纷时，普通职场人往往处于极端的信息不对称状态：

**法律知识劣势**：公司背后是专业的法务团队，而你可能连 N、N+1、2N 的区别都搞不清楚。

**谈判心理压力**：HR 往往利用话术施压，普通人在焦虑和恐惧下极易做出不利于自己的决策。

**证据意识薄弱**：不知道哪些沟通记录可以作为证据，什么时候该录音，什么时候该签字。

我做这个项目，就是希望通过 AI 将复杂的法律逻辑和谈判策略“工具化”，为每一个普通的职场人提供平等的“法律武装”。

## ✨ 3. Agent做了哪些差异化设计

如果你只是在 ChatGPT 里问“被裁了怎么办”，你得到的通常是泛泛而谈。本项目基于 Dify 工作流 开发，具备以下普通对话产品不具备的硬核卖点：

**一对一深度定制化建议**：

拒绝模版化：Agent 会根据你实时输入的岗位性质、入职时间、城市政策等，识别你当下的离职协议陷阱。
动态策略：它不是给出一套死板的建议，而是根据你和 HR 谈到的“最新价码”实时调整后续的谈判筹码。

**多模态深度解析**：

支持上传 劳动合同照片、解除协议截图、录音转文字记录。AI 会自动扫描其中的关键条款和潜在陷阱，比肉眼更敏锐。

**精准的状态管理**（State Management）：

**长效记忆**：Agent 能够时刻锁定你的谈判进度、历史报价和关键诉求，不会因为对话变长而“断片”或忘记之前的承诺。

**法律知识库与幻觉约束**：

拒绝瞎编：通过 RAG（检索增强生成）技术，强制 AI 优先检索真实的法律法规条文。

**严谨计算**：内置专业提示词框架，最大限度减少 AI 在赔偿数额计算上的“幻觉”现象，确保每一分钱都有据可依。

## 🛠️ 4. 应该怎么用？

第一步：导入 DSL
下载本仓库中的 AI裁员谈判专家.yml。

登录 Dify 控制台，点击“创建应用” -> “导入 DSL 文件”。

第二步：关键配置
自备 API Key：导入后，请在 Dify 设置中配置你自己的模型供应商（如 OpenAI、DeepSeek 或硅基流动）。

关联知识库：

在 Dify 中新建一个知识库，上传当地《劳动合同法》或典型案例。

在工作流的“知识检索”节点中重新关联该知识库。

插件授权：本项目使用了 siliconflow 插件，请确保在 Dify 插件页面已授权。
